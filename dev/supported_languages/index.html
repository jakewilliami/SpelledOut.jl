Supported Languages

Current languages to choose from are:

  • English
    • Variants
      • :en_US (aliases: :en)
      • :en_UK (aliases: :en_GB; equivalencies: :en_NZ, :en_AU)
    • Dictionaries supported include :modern, :british, and :european, and default to the former.
  • Spanish (:es)
  • Portuguese
    • Variants
      • (:pt_BR)
      • (:pt)
